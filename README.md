
بسم الله الرحمن الرحيم

برنامج رمضان كريم

يعرض أوقات الصلاة  حسب الموقع 

يعرض التاريخ الهجري والميلادي الحالي

يقدم قسم لمتابعة إنجاز المهام والعبادات اليومية

يحتوي على ملخص أذكار الصباح والمساء وأدعية مختارة

يوفر متابعة يومية للورد القرآني ونسبة التقدم في القراءة

يتضمن قائمة للعبادات مثل الصلاة في وقتها والذكر والصدقة

يحتوي على حاسبة زكاة لحساب المبلغ الواجب إخراجه

يعرض حكمة يومية لتحفيز المستخدم روحياً

يقدم لوحة تحكم تساعد على تنظيم العبادات خلال رمضان
